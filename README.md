## ✨ Using Your New Console

When you launch MobaXterm for the first time, a dark window appears. This is your terminal.

1. Click the "Session" button in the top left corner.
2. Select "SSH" from the menu.
3. Type the IP address or host name of the computer you want to reach into the "Remote host" box.
4. Check the "Specify username" box if your server requires a specific account name.
5. Click "OK" to begin the connection.

A prompt asks for your password. Type the password and press Enter. The characters might not appear on the screen for security reasons. This is normal behavior for terminal software.

## 📂 Managing Remote Files

*   **Edit:** Right-click a file to choose the "Edit" option. This opens the file in a built-in text editor that saves changes directly to the server.
